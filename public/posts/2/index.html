💡 核心要点详解 Link to heading

1️⃣ 了解你的合法权益 Link to heading

根据《劳动合同法》，劳动者享有以下基本权利：

💰 获得报酬权

公司必须按时足额支付工资，不得拖欠

🏖️ 休息休假权

享有法定节假日、带薪年假等权利

🛡️ 社保缴纳权

公司必须为员工缴纳五险一金

💵 经济补偿权

被辞退时有权获得相应经济补偿

2️⃣ 被辞退时的应对策略 Link to heading

❌ 错误做法： Link to heading

  • ❌ 冲动离职，放弃维权
  • ❌ 在公司施压下匆忙签署离职协议
  • ❌ 没有保留任何证据
  • ❌ 情绪化处理，与领导发生冲突

✅ 正确做法： Link to heading

第一步：保持冷静，不要立即签字

  • 📄 仔细阅读公司给出的所有文件
  • ⏰ 要求给予充分考虑时间
  • 📝 要求公司出具书面的辞退通知
  • 🤐 不要轻易表达"我同意"或"我辞职"

第二步：全面收集证据

📂 必备证据清单：

  • 劳动合同：原件或扫描件
  • 工资证明：工资条、银行流水（至少12个月）
  • 工作记录：工作邮件、微信聊天记录、工作群聊记录
  • 考勤记录：打卡记录、请假记录
  • 工作成果：项目文档、工作报告
  • 录音录像：与HR或领导的谈话录音（合法取证）
  • 公司通知：辞退通知、调岗通知等书面文件

第三步：了解补偿标准

辞退情况 补偿标准 说明
公司主动辞退（无过错） N+1 个月工资 N为工作年限，+1为代通知金
违法辞退 2N 个月工资 可要求继续履行合同或赔偿金
协商解除 协商确定 一般不低于N的标准
员工主动辞职 无补偿 除非公司有过错

💡 计算公式：
N = 工作年限

  • 满1年算1个月工资
  • 6个月以上不满1年算1个月
  • 不满6个月算半个月工资

工资标准 = 离职前12个月平均工资（包括奖金、津贴等）

4️⃣ 常见陷阱及应对 Link to heading

⚠️ 公司常见套路

陷阱1：公司让你主动辞职

目的：避免支付经济补偿金

应对：坚持不主动辞职，要求公司出具正式辞退通知。如公司施压，可以录音取证。

陷阱2：调岗降薪逼你走

目的：通过不合理调岗让你主动离职

应对：书面拒绝（邮件或微信），明确表示不同意，保留证据，可申请劳动仲裁。

陷阱3：不交或少交社保

目的：节省成本

应对：向社保部门投诉，要求补缴。这是违法行为，公司必须补缴并可能面临罚款。

陷阱4：拖欠或克扣工资

目的：减少支出

应对：保留工资条、银行流水等证据，可向劳动监察投诉或申请仲裁，要求支付拖欠工资及经济补偿。

🛡️ 预防措施 Link to heading

入职前： Link to heading

  • ✅ 仔细审查劳动合同条款，特别是试用期、薪资、违约金等
  • ✅ 了解公司背景、口碑和经营状况
  • ✅ 明确薪资结构、发放时间、福利待遇
  • ✅ 确认社保缴纳基数和比例

工作中： Link to heading

  • ✅ 定期保存工作记录和成果
  • ✅ 保留工资条和银行流水（至少12个月）
  • ✅ 重要沟通尽量书面化（邮件、微信）
  • ✅ 关注社保缴纳情况，定期查询

离职时： Link to heading

  • ✅ 要求公司出具正式的离职证明
  • ✅ 核对经济补偿是否计算正确
  • ✅ 办理社保转移和公积金提取手续
  • ✅ 交接工作要留书面记录

🎯 总结 Link to heading

职场维权不是"闹事"，而是维护自己的合法权益。关键在于：

  1. 懂法：了解《劳动合同法》等相关法律法规
  2. 留证：平时注意保留工作记录、工资流水等证据
  3. 理性：用合法途径维权，不要冲动行事
  4. 坚持：维权可能需要时间，但正义终将到来

💪 记住：法律是保护劳动者的，只要你站在法律这边，就不用害怕！
